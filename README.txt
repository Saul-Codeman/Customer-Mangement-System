## Purpose
To demonstrate advanced java concepts. The Scheduler application is designed to efficiently manage customer
and appointment data for businesses. It allows users to easily create, update, and delete customer information,
as well as schedule, modify, and cancel appointments.
## Functionality of the program
- Using a database to execute queries on the application
- Add, modify, remove customers and appointments with constraints
- Generate login information
- Generate confirmation messages
- Exception handling
- TimeZone support
- JavaFX and UI
- Git and Github
- At least 2 lambda expressions (AddCustomerController and Appointments model)

## Directions to run program
- Ensure that Java SE 17.0.1 and JavaFX 17.0.1 are installed on your system.
- Clone or download the project repository from the provided source control link (e.g., GitHub or WGU).
- Open the project in your preferred IDE (e.g., IntelliJ IDEA, Netbeans, etc.).
- Ensure that your IDE is configured to use the correct JDK and JavaFX versions.
- Add the JavaFX libraries to the project if they are not already included (usually located in the javafx-sdk/lib folder).
- Locate the main class of the application, usually containing a public static void main(String[] args) method.
- Run the main class, and the application should launch successfully.
